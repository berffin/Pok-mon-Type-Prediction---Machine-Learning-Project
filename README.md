A machine learning project that predicts Pokémon types based on their base stats using Logistic Regression.

## Project Demo
![Confusion Matrix](confusion_matrix.png)

## Features
- **Type Prediction**: Predicts Pokémon type from 6 base stats
- **Model Evaluation**: Comprehensive performance analysis
- **Feature Importance**: Identifies most important stats
- **Visualizations**: Confusion matrix and feature importance charts
- **Real-world Testing**: Test with famous Pokémon examples

## Results
- **Accuracy**: XX.XX% on test set
- **Best Feature**: HP (most important for prediction)
- **Dataset**: 800+ Pokémon from 8 generations
- **Model**: Logistic Regression with 5-fold cross-validation
